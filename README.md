# AI 에이전트 스튜디오 프로토타입

AI 에이전트 기능 정책 기획서를 기반으로 한 프로토타입 애플리케이션입니다.

## 주요 기능

### 1. 에이전트 관리
- 에이전트 목록 조회
- 에이전트 생성/수정/삭제
- 라이선스 기반 생성 제한 (스탠다드 * 1 + 프리미엄 * 5)

### 2. 에이전트 설정
- **기본 정보**: 이름, 프로필 이미지, 설명, 조직 정보
- **데이터 설정**: 
  - 언어 모델 선택
  - 검색 그라운딩 이용 여부
  - 참조 데이터 (업무/위키/드라이브) - 다이얼로그를 통한 선택
  - 상세 지침
  - 외부 연동 (APP 연동, MCP 서버 연동)
- **트리거 설정**: 
  - 자연어 트리거
  - 키워드 트리거
  - 명령어 트리거
- **행동 설정**: 
  - 트리거별 여러 행동 설정 가능
  - 메시지 답변 / 알림 전송 / 둘 다 선택 가능
  - 자동 답변 및 승인 대상 설정
  - 알림 스킬 설정 (Webhook URL, 메시지 템플릿, 맥락 포함)
- **권한 설정**: 에이전트 관리자, 이용 권한, 메신저 대화방 이용 권한

### 3. 활동 이력 조회
- 답변 이력
- 트리거 발동 이력
- 참여 대화방 확인

## 기술 스택

- React 18
- TypeScript
- Vite
- React Router
- Lucide React (아이콘)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

개발 서버 실행 후 브라우저에서 다음 URL로 접속하세요:
- **http://localhost:5173**

### 주요 페이지
- **에이전트 목록**: http://localhost:5173/
- **에이전트 생성**: http://localhost:5173/agents/create
- **에이전트 수정**: http://localhost:5173/agents/:id/edit
- **활동 이력**: http://localhost:5173/agents/:id/activity

### 빌드

```bash
# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 프로젝트 구조

```
src/
├── components/          # 재사용 가능한 컴포넌트
│   ├── Layout.tsx      # 레이아웃 컴포넌트
│   └── AgentForm.tsx   # 에이전트 생성/수정 폼
```

## 주요 화면

1. **에이전트 목록**: 생성된 모든 에이전트를 테이블 형태로 표시
2. **에이전트 생성/수정**: 탭 기반 설정 폼 (기본 정보, 데이터 설정, 트리거 설정, 행동 설정, 권한 설정)
3. **활동 이력**: 에이전트의 답변 및 트리거 발동 이력 조회

## 주요 특징

- **다중 행동 설정**: 각 트리거에 대해 여러 행동(메시지 답변, 알림 전송 등)을 설정할 수 있습니다
- **외부 연동 통합**: APP 연동과 MCP 서버 연동을 하나의 "외부 연동" 섹션에서 관리합니다
- **참조 데이터 다이얼로그**: 업무/위키/드라이브 데이터를 다이얼로그를 통해 선택적으로 추가할 수 있습니다
- **트리거 유형**: 자연어, 키워드, 명령어 트리거를 지원합니다
